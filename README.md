# The Humans

Десктопное приложение для хранения и структурирования информации о людях.

## Возможности

- Группировка контактов: Работа, Друзья, Случайные, Родственники + виртуальная группа «Все»
- Подробные карточки людей с фото, контактами и расширенным профилем
- Поиск по имени и заметкам
- Сортировка: по ФИО, дате рождения, ближайшему ДР, дате добавления, надёжности
- Экспорт дней рождения и событий в `.ics` (iCalendar) для Google Calendar, Outlook, Apple Calendar
- Уведомления о ближайших событиях через системный трей
- Управление событиями (годовщины, важные даты) с настройкой напоминаний
